Bienvenido a la App Educativa del Clima

Esta aplicación te permite consultar el clima actual de cualquier ciudad ingresando su nombre. Utiliza la API de OpenWeatherMap para obtener la información meteorológica más reciente.

Para comenzar, navega a la sección "Time" en la barra de navegación y completa el formulario con el nombre de la ciudad que deseas consultar.

La aplicación mostrará la temperatura actual, la latitud y longitud de la ciudad, y otros datos relevantes.

¡Esperamos que disfrutes usando esta aplicación y que te sea de utilidad!
